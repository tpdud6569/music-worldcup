Music Worldcup

Music Worldcup is a web application that allows users to log in with their Google account and create a music worldcup using their own YouTube playlists or liked videos.

The application uses the YouTube Data API in read-only mode. Retrieved data is used temporarily during the session and is not stored permanently.
